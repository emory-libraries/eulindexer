Documentation for The EULIndexer |release|
==========================================

EULIndexer is an application based in Python_ and Django_ for indexing data from
a Fedora-Commons_ Repository into a Solr_ index.

.. _Python: http://www.python.org/
.. _Django: http://www.djangoproject.org/
.. _Fedora-Commons: http://fedora-commons.org/
.. _Solr: http://lucene.apache.org/solr/
